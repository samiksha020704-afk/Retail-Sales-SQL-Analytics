# Retail-Sales-SQL-Analytics

## Project Overview
This project analyzes retail sales data using SQL and MySQL Workbench.

The project includes:
- Data exploration
- Sales analysis
- Customer analysis
- Product analysis
- Advanced SQL queries
- Business insights generation

## Tools Used
- SQL
- MySQL Workbench
- GitHub

## Key SQL Concepts Used
- JOINS
- GROUP BY
- ORDER BY
- Aggregate Functions
- Subqueries
- CASE Statements
- LIMIT
- Data Filtering
